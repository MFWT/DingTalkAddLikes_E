# DingTalkAddLikes_E
**钉钉直播刷赞**  

**内卷时代的产物**

## 使用方法
### 太长不看版
1.用Fiddler或者别的什么工具，获取直播间UUID（https://hd.mmstat.com/taobaolive.video.x  GET请求参数中LiveUuid的值）  
（钉钉需要将代理设置为抓包工具提供的代理地址。抓包工具需开启解密HTTPS功能）  

2.将UUID填入指定位置，并设置好数量

3.开始
